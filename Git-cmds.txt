REACT PROJECT GIT COMMANDS :-

1️⃣ Repo initialize karna (Agar naye folder me Git shuru kar rahe ho)
git init


2️⃣ Remote repository add karna (Sirf pehli baar)
git remote add origin <repo-URL>
Example:
git remote add origin https://github.com/Capten05ast/Stack_Stock_Store.git


3️⃣ File changes stage karna
git add .


4️⃣ Commit create karna
git commit -m "Your commit message"


5️⃣ GitHub changes fetch karna (Agar repo me pehle se kuch code hai)
git fetch origin


6️⃣ Apna local code aur GitHub ka code merge/rebase karna
(Merge ke badle rebase use karna zyada clean hota hai)
git pull --rebase origin main


7️⃣ Final push karna
git push -u origin main


8️⃣ Agar remote origin galat set ho gaya ho (Delete + naya add)
git remote remove origin
git remote add origin <repo-URL>

DIST FOLDER :-
for deploying your website on render you may create dist folder which
will convert you all react code into HTML, CSS and JS. cmd for creating
dist folder : npm run build




RENDER :-
If you are going to publish this website on render, then keep these 
things in mind :-
> choose static site in Add new
> your build command should be : cd frontend && npm install && npm run build
> your directory should be : frontend/dist
> Now simply deploy
> Why we did this becoz package.json file is inside the frontend folder, 
  which means dist and package.json should be in one folder
